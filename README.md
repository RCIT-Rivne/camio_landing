# CamIO Landing Page

Односторінковий лендінг CamIO для презентації тактильної навігації Рівненського фахового коледжу інформаційних технологій спільно з The Smith-Kettlewell Eye Research Institute.

## Як переглянути

```bash
# у корені репозиторію
xdg-open index.html
```

Або розгорніть сторінку на будь-якому статичному веб-сервері (Apache, Nginx, GitHub Pages, Netlify). Додаткових залежностей немає.

## Структура

- `index.html` — доступна, адаптивна сторінка з усіма секціями, стилями та мінімальним JavaScript (мобільне меню та галерея).
- `README.md` — ця інструкція.

## Перемикання мов

- У хедері є перемикач UA/EN. За замовчуванням сторінка відкривається українською.
- Після кліку перекладаються всі текстові блоки, alt-описи зображень, aria-label елементів та метадані.
- Перемикач доступний з клавіатури й оновлює атрибут `lang` у `<html>`.

## Брендові стилі

- Основний шрифт тексту — Open Sans (Google Fonts), заголовки й логотип використовують Gilroy Extra Bold (підтягнеться, якщо встановлено локально; є fallback на Open Sans).
- Палітра побудована на кольорах коледжу: фіолетовий `#600CEB`, жовтий `#F9DB74`, оранжевий `#E36756`, блакитний `#5FBDF7`, додаткові відтінки для фону та бордерів.
- Hero-блок отримав градієнт, картки та кнопки мають акцентні бордери/заливки, фокуси виділяються жовтим `#F9DB74` для зручної клавіатурної навігації.

## Примітки

- Замініть плейсхолдери `placeholder.jpg` на реальні зображення та оновіть атрибути `alt` за потреби.
- Оновіть `mailto:contact@example.com` та інші коментарі в коді на фактичні контактні дані й посилання.
- У секції відео замініть `VIDEO_ID` на справжній ідентифікатор YouTube.
